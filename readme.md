# AAWEB Product Image Hover Swap for Woocommerce

AAWEB Product Image Hover Swap for Woocommerce adds a second-image hover swap effect to WooCommerce product cards, Elementor product grids, ShopEngine product lists and WooCommerce block-based catalogs.

## Requirement

This plugin requires WooCommerce.

## Features

- WooCommerce product loop support
- Elementor product grid support
- ShopEngine product list support
- WooCommerce Blocks support
- AJAX DOM fallback
- MutationObserver support
- Desktop-only mode
- Mobile-safe behavior
- Translation-ready strings
- Clean uninstall cleanup
